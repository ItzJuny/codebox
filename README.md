# codebox
一些代码合集
- kaldi_feats.py 借用KALDI来生成MFCC CMVN特征文件。[Kaldi安装见步骤七](https://blog.csdn.net/sleepinghm/article/details/117631606)
- pcm2wav.py pcm文件转换wav文件
- wav2pcm.py wav文件转换pcm文件
- pcmsplit.py 分割pcm文件
- wavsplit.py 分割wav文件
- readnpy.py 读取npy文件
- pdf_process.py 转换pdf和jpg
- backup.py 简易备份两个目录中的文件
